# GovAssist QA Pilot
 
A mini government-services AI assistant (RAG + agentic chatbot) with a full QA suite.
Built as a portfolio project
 
## Stack
- App: Node.js + Express
- LLM: Ollama (llama3.2)
- UI Tests: Playwright
- LLM Eval: Promptfoo
- Performance: k6
- Accessibility: axe-core
- CI/CD: GitHub Actions
